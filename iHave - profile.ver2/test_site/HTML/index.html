    ABOUT US

    Pilates Studio iHavëは、洗練された清潔感あふれる完全プライベート空間で、心と身体のバランスを整えられる、安心して通えるスタジオを目指しています。

    理想のボディラインを追求したり、姿勢の改善を目指したりと、オーダーメイドプログラムを構成し、
    忙しい毎日の中で、自分自身と向き合う大切な時間を過ごしてほしい——そんな想いからスタジオを立ち上げました。

    ”更なる美の追求”をテーマに、自分の身体の動きや感覚に集中し、自己理解を深め、
    体調や姿勢の改善を通じて、自己肯定感と日常の充実感が高まることを目指しています。

    完全個室で丁寧にサポートいたしますので、初めての方でも安心してご参加いただけます。
    既往歴がある方や聴覚などに不安があるお客様でも安心。医療根拠に基づいた指導でサポートいたします。

    STUDIO

    当スタジオは、心と身体の調和を追求するための特別な空間です。

    大きな窓から差し込む自然光が室内全体を優しく照らし、
    明るく開放的な雰囲気を演出しています。
    自然光がもたらすリズムと活力が、心地よくトレーニングに集中できる環境を整えます。

    スタジオ内には厳選された植物が点在し、緑の癒しと新鮮な空気に包まれることで、自然とのつながりを感じながら、より深いリラックス効果を得られます。

    空間全体は、モダンなインテリアに木の温もりを調和させた、
    洗練されたデザイナーズ仕様。
    自然光、植物、デザインが融合するスタジオで、自分自身と向き合う豊かな時間をお過ごしください。
